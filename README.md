# Speech Recognition Using Deep Learning
### Models that are already supported
1. [Conformer: Convolution-augmented Transformer for Speech Recognition](https://arxiv.org/abs/2005.08100)
2. [E-Branchformer: Branchformer with Enhanced merging for speech recognition](https://arxiv.org/abs/2210.00077)

### Datasets that are already supported
1. [LibriSpeech](https://www.openslr.org/12)
2. [Common Voice](https://commonvoice.mozilla.org/en/datasets)
3. [Aishell-1](http://www.openslr.org/33/)
4. [Aishell-2](http://www.openslr.org/93/)
5. [THCHS-30](http://www.openslr.org/18/)
